How to Return a Book

To successfully return a book, please follow these steps:

  • 1. Enter Your User ID: This is the unique ID you used when borrowing the book.
  • 2. Enter the Book ISBN: Make sure you enter the exact ISBN of the book you borrowed.
  • 3. Click "Return Book": Once you've filled in both fields, click the button to complete the return.
